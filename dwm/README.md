## Install

```shell
sudo make clean install
```

## Set

You can edit the file [config.h](./config.h)
to build your own dwm, there are 3 themes you can change.

+ [gruvbox](./themes/gruvbox.h)
+ [onedark](./themes/onedark.h)
+ [tokyonight](./themes/tokyonight.h)
